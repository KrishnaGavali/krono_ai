# ü§ñ Krono AI: The Conversational Triage Assistant

Krono AI is a **Conversational AI Productivity Tool** designed to eliminate the cognitive burden and friction of manual task entry, especially for students managing chaotic group work. It transforms messy, unstructured communication into actionable, structured data. The project is an **EdTech** solution built to showcase expertise in full-stack AI orchestration and real-time database management.

---

### Key Workflow

1.  **Messy Input:** A user sends a complex, conversational message (or voice note) via **WhatsApp** (or a simulation input for the MVP) containing multiple tasks, due dates, and contexts.
2.  **AI Triage:** The input is immediately routed to the **Gemma 3 LLM**. The LLM's prompt is engineered to parse the raw text and return a structured **JSON** object, identifying the task, priority, owner, and due time.
3.  **Data Persistence:** The structured JSON is saved to the **Appwrite** real-time database. Appwrite's backend efficiently manages the data schema, authentication, and user-specific task lists.
4.  **Action & Display:** The frontend **Next.js** **Intelligent Dashboard** (or "Plan Page") uses an Appwrite Real-time Listener to instantly update. It shows a prioritized **"What's Important Now"** list and pushes final, clean events/tasks to the user's **Google Calendar**.
